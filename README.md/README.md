# 🌱 Weather & Crop Recommendation System

An AI-powered agriculture recommendation system using Machine Learning and Data Analytics.

## Features
- Crop prediction using weather and soil data
- Random Forest ML model
- Streamlit dashboard
- Data visualization
- Crop analytics

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib

## How to Run

```bash
streamlit run app.py
